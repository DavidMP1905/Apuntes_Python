# Operadores en asignación

Actúan directamente sobre la variable actual modificando su valor.

## Suma en asignación

```python
a = 5
a += 5  
a
```

## Resta en asignación

```python
a = 5
a -= 10  
a
```

## Producto en asignación

```python
a = 5
a *= 2  
a
```

## División en asignación

```python
a = 5
a /= 2  
a
```

## Módulo en asignación

```python
a = 5
a %= 2
a
```

## Potencia en asignación

```python
a = 5
a **= 3
a
```
